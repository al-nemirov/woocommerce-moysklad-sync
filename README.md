# WooCommerce МойСклад Sync

Синхронизация заказов WooCommerce с [МойСклад](https://www.moysklad.ru/) через API Remap 1.2.

## Возможности

- Автоматическая отправка заказов в МойСклад при достижении выбранного статуса WC
- Ручная отправка кнопкой из карточки заказа
- Заказы создаются как **черновики** (`applicable: false`) — не проводятся автоматически
- Автоматическое создание контрагентов (поиск по email, создание если не найден)
- Сопоставление товаров по артикулу (SKU → article в МойСклад)
- Товар по умолчанию для позиций без совпадения
- Кнопка «Пересоздать» для повторной отправки
- Совместимость с HPOS (WooCommerce 7+)
- Автоматическая миграция настроек с плагина Яндекс Доставка (если ранее использовался)

## Установка

1. Скопируйте папку `woocommerce-moysklad-sync` в `wp-content/plugins/`
2. Активируйте плагин
3. Перейдите в **WooCommerce → МойСклад**
4. Введите логин и пароль МойСклад
5. Выберите статус для автоотправки
6. Опционально: укажите UUID организации и товара по умолчанию

## Настройки

| Параметр | Описание |
|----------|---------|
| Включить | Вкл/выкл синхронизации |
| Логин | Email аккаунта МойСклад |
| Пароль | Пароль аккаунта |
| Статус отправки | Статус WC при котором заказ уходит в МС |
| ID организации | UUID организации (необязательно — берётся первая) |
| Товар по умолчанию | UUID товара для позиций без совпадения по артикулу |

## Мета-бокс в заказе

В карточке заказа WooCommerce появляется панель «МойСклад»:
- **Не отправлен** — кнопка «Отправить в МойСклад»
- **Отправлен** — ссылка на заказ в МойСклад + кнопка «Пересоздать»
- **Ошибка** — текст ошибки + кнопка повторной отправки

## Требования

- WordPress 5.8+
- WooCommerce 5.0+
- PHP 7.4+
- Аккаунт МойСклад с доступом к API
